# CV – Felipe Fazio da Costa

Personal CV website hosted on GitHub Pages.

**Live:** https://felipefaziodacosta.github.io/CV_Felipe_Fazio

## Features
- Light (minimalist) & Dark mode with system preference detection and persistence
- Portuguese / English language toggle
- Smooth scroll-reveal animations
- PDF download button
- Fully responsive

## Structure
```
index.html          ← single-page CV
css/style.css       ← all styles with CSS custom properties for theming
js/main.js          ← theme toggle, language switch, scroll animations
assets/             ← place cv_felipe_fazio.pdf here
```

## Deploy (GitHub Pages)
1. Go to the repository on GitHub → **Settings → Pages**
2. Source: `Deploy from a branch` → branch `main`, folder `/ (root)`
3. Save. The site will be live in ~1 minute at the URL above.
